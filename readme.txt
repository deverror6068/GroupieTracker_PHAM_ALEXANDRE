Projet Groupie Tracker

Objectifs: 

Réaliser un site Web hébergé en
GoLang qui affiche et illustre les
données récupérées depuis une des
APIs proposées.



Temps de réalisation: 
16 jours 

Organisation des dossiers: 

Il y a un dossier src qui contient le code  et un dossier doc qui contient le dev journal 

Recommandations : 

lancez le programme puis tapez local/home sur votre navigateur pour accéder au serveur 



Fonctionnement du serveur: 
 
La page rechercher récupère la valeur entrée par l' utilisateur (elle effectue ensuite une requete url)  et la stocke dans un fichier txt qui est lu et récupéré par la page résultats qui effectue ensuite la requete


Système de gestion d'erreur personnalisé: 

le système de gestion d'erreur est personnalisé pour afficher de manière plus ergonomique et précise les erreurs avec un code d'erreur qui lui est propre


Code d'erreur : 

code 15: erreur lors  de l'ouverture de fichier 
code 16 : erreur lors la lecture du fichier 
code 17 : erreur lors de la suppression du fichier 
code 18 : erreur  lors de la création du fichier
